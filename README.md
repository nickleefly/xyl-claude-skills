# XYL Claude Skills

Collection of Claude Code skills

## Skills

- **x-bookmarks**: Export X/Twitter bookmarks to markdown using the bird CLI
- **substack**: Publish newsletter articles to Substack as drafts from Markdown

## Claude Code Plugin

Install from marketplace:

```bash
/plugin marketplace add nickleefly/xyl-claude-skills
/plugin install x-bookmarks@xyl-marketplace
/plugin install substack@xyl-marketplace
npx skills add x-bookmarks@xyl-marketplace
npx skills add substack@xyl-marketplace
```
